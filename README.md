## 📚 About Time_Warp_II

Time_Warp_II is a special version of the original Time Warp Classic. It is made just for the TempleCode language, which combines BASIC, PILOT, and Logo. This mix makes learning to code simple and fun. You can write programs using turtle graphics, see your code with colors thanks to syntax highlighting, and enjoy a classic computer style.

This program works as an educational IDE. IDE means Integrated Development Environment. It is a tool where you can write, check, and run your code in one place. Time_Warp_II aims to make programming easier for beginners and hobbyists by providing a clear and visual way to learn.

## 🛠️ How to Use Time_Warp_II

Once the software is running, you will see a workspace with several features:

- **Code Editor**: This is where you write your programs. It highlights commands in color to help you read and catch mistakes faster.  
- **Turtle Graphics Window**: This window shows pictures created by your code, using the famous turtle graphics system.  
- **Run Button**: Use this button to start your program and see the output or graphics.  
- **Stop Button**: Stops a running program anytime you want to end it early.  
- **File Menu**: Lets you create, open, save, and close your code files easily.

You can start by writing simple commands in the TempleCode language, then run your program to see how the turtle moves or how the code behaves.

---

## 💾 Saving and Loading Files

To keep your work, use the **File** menu:

- **New**: Start a new program from scratch.  
- **Open**: Load an existing `.temple` or `.txt` file you saved before.  
- **Save**: Save your current program. If it’s new, you will be asked to name the file.  
- **Save As**: Save a copy under a different name or location.

Files saved this way can be opened any time in Time_Warp_II or other basic text editors.

## ⚙️ Common Features Explained

### Turtle Graphics  
You can move a turtle cursor on the screen that draws lines. Commands like `FORWARD`, `LEFT`, and `RIGHT` control its movement. This helps you learn geometry and programming together.

### Syntax Highlighting  
The editor shows keywords, commands, and numbers in different colors. This makes understanding your code faster.

### Retro-Style Interface  
Time_Warp_II uses old-style computer colors and fonts to give a nostalgic feel, similar to early home computers.
